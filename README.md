## Papar Information
- Title:  `Filament Example`
- Authors:  `AYMAN NAGY AHMED`


## Install & Dependence
- php >= 8.0
- laravel 10


## Use
- to run
  ```
$ composer install

$ php artisan migrate

$ php artisan storage:link

$ php artisan db:seed

$ php artisan serve
  ```
